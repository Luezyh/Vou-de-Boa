# 📍 Vou De Boa

Aplicação web desenvolvida para ajudar usuários a encontrar locais de lazer e entretenimento próximos de sua localização de forma rápida, prática e intuitiva.

## 🎯 Objetivo

O projeto **Vou De Boa** tem como objetivo facilitar a busca por estabelecimentos e pontos de interesse próximos, permitindo que os usuários encontrem opções de lazer sem precisar consultar diversos aplicativos.

A plataforma exibe locais como:

* 🍽️ Restaurantes
* ☕ Cafeterias
* 🍺 Bares
* 🌳 Parques
* 🛍️ Shoppings
* 🎉 Eventos

Além disso, são apresentadas informações como imagens, avaliações, categorias e distância até o local.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## ✨ Funcionalidades

* Busca de locais próximos.
* Interface responsiva para dispositivos móveis.
* Exibição de informações dos estabelecimentos.
* Organização por categorias.
* Visualização de distância entre usuário e local.
